📁 FILE REFERENCE
─────────────────

combined_simple_teleop_real_logger.py (410 lines)
  Main script: live teleoperation + HDF5 logging
  Usage: python combined_simple_teleop_real_logger.py [options]
  Default: records with HDF5 logging enabled

robot_pose_controller.py (420 lines)
  Controller class: send any 6D pose + hand commands  
  Usage: python robot_pose_controller.py --pose X Y Z Rx Ry Rz
  Also imported by replay_hdf5.py

replay_hdf5.py (370 lines)
  Replay recorded trajectories from HDF5
  Usage: python replay_hdf5.py file.hdf5 [options]
  Features: homing phase, smooth motion, timing preservation

logs/
  Directory: HDF5 recordings (auto-created)
  Format: teleop_YYYYMMDD_HHMMSS.hdf5
  Size: ~5-10 MB per minute of recording

🚀 COMMAND QUICK REFERENCE
───────────────────────────

Record:
  python combined_simple_teleop_real_logger.py

Replay (real-time):
  python replay_hdf5.py logs/teleop_*.hdf5

Replay (half speed):
  python replay_hdf5.py logs/teleop_*.hdf5 --speed 0.5

Inspect file:
  python replay_hdf5.py logs/teleop_*.hdf5 --dry-run

Send single pose:
  python robot_pose_controller.py --pose 0.3 0.0 0.2 0.0 0.0 0.0

Full help:
  python combined_simple_teleop_real_logger.py --help
  python replay_hdf5.py --help
  python robot_pose_controller.py --help
